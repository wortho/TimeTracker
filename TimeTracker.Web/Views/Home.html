TimeTracker

Enables you to register the time you spend on your projects, so that you can create correct invoices for your customers. Enables you to an overview of your time registrations, so that you can create correct invoices for my customers.

  1. Choose a Customer from the list of Customers
  2. Choose a Projects Link to view the list of existing projects for the Customer
  3. Choose a Time Entries Link to view the list of existing time entries for the Project
  4. Click on the Register Time Button to add a new time entry for the Project
